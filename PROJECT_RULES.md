# QUY TẮC DỰ ÁN HOTEL APP 2026 
- Gemini là đại diện kỹ thuật duy nhất của Chủ dự án. Trae thực hiện theo Prompt của Gemini. 
- Tuyệt đối không tự ý thêm/sửa/xóa cột trong Database. Mọi thay đổi phải qua Gemini. 
- Code Frontend phải khớp 100% với file `DATABASE_SCHEMA.md`. 
- Quy trình nghiệm thu: Trae làm -> Chủ dự án Test -> Gemini soạn tóm tắt -> Trae dán vào bộ nhớ.
